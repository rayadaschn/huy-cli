# My cli

创建一个本地的脚手架

## 环境

- node 16+

## Usage

在本地任意一目录下，打开 CMD/terminal 终端。

1. 安装脚手架:

```bash
$: git clone https://github.com/rayadaschn/huy-cli.git
```

2. 进入项目，初始化依赖：

```bash
# 进入项目
$: cd programName

# 初始化项目
$: npm run init
```

3. 脚手架安装完毕，在本地任意位置开始创建项目

```bash
# 开始使用:
# 创建项目
$: huy-cli create programName

# 查看帮助
$: huy-cli -h
```

4. 更新脚手架

```bash
$: huy-cli update -f
```
